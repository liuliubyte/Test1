# 🎓 学习通全能 AI 做题助手 (ChaoXing AI Assistant)
![Author](https://img.shields.io/badge/author-Hutch-orange.svg)

基于大语言模型（支持智谱 GLM / DeepSeek）的 Tampermonkey（油猴）自动化脚本，专为解决复杂网页嵌套环境下的题目抓取与自动作答而设计。告别繁琐的弹窗，享受沉浸式的自动化做题体验！

✨ **UI Design by Hutch**

---
**必读**(智谱可使用官方免费的glash4 flash模型，但速度较慢质量较差，这里建议使用DeepSeek版本，速度更快更稳定，且V4-flash模型性价比非常高，最后一点，代码里记得添加你的API(只要在一个位置添加即可)，代码不够完美精简，欢迎各位共同升级)

## 🌟 核心特性 (Features)

* 🚀 **穿透式题目抓取**：独创的递归 iframe 穿透算法，无论网页嵌套多少层，都能一键将所有题目纯文本无损抓取，方便离线复习或归档。
* 🎯 **客观题高容错秒杀**：
    * **Token 极限压缩**：采用高密度 JSON 格式传输请求，最大化利用单次 API 的 Token 额度。
    * **断点紧急抢救机制**：面对超大题量导致 AI 回复被截断（JSON 不完整）时，系统会自动触发 `lastIndexOf('}')` 抢救机制，挽回已生成的答案，并在实时日志中精准提示。
* ✍️ **主观题逐题精答**：彻底剥离 JSON 限制！对简答题、填空题采用“逐题深度思考”模式，自动定位富文本编辑器或隐藏文本框并注入答案，长篇大论再也不会报错。
* 💻 **沉浸式极客面板 (Hutch UI)**：
    * **自由拖拽与折叠**：现代化悬浮卡片设计，支持全屏自由拖动，不遮挡做题视线。
    * **实时终端日志窗**：用底部滚动日志完全替代了烦人的系统 `alert` 弹窗，执行状态（正在抓取、抢救触发、执行完毕）一目了然。
    * **强制挂载防丢机制**：内置 DOM 轮询守护进程，确保无论页面如何卡顿重载，控制面板 100% 稳定呈现。

## 🛠️ 安装指南 (Installation)

1. **安装浏览器扩展**：请先在浏览器（Chrome / Edge / Firefox）中安装 [Tampermonkey (油猴)](https://www.tampermonkey.net/) 插件。
2. **添加脚本**：
    * 点击油猴图标 -> **添加新脚本 (Create a new script)**。
    * 将本项目中的完整代码复制并粘贴到编辑器中。
    * 按 `Ctrl + S` (或 `Cmd + S`) 保存。
3. **配置 API Key (重要 ⚠️)**：
    * 在代码的第 16 行左右找到 `const API_KEY = "这里填入你的API_KEY";`。
    * 替换为您申请的真实 API Key（智谱或 DeepSeek）。

## 🚀 使用说明 (Usage)

1. 打开任意学习通测验/考试页面。
2. 页面加载完毕后，左侧会自动弹出 **✨ Q&A Assistant** 悬浮控制面板。
3. 面板操作指南：
    * **【🚀 0. 穿透抓取题目】**：仅抓取并打印全卷题目到 F12 浏览器控制台（不会自动答题）。
    * **【🎯 1. 秒杀客观题】**：自动筛选页面中的单选、多选、判断题交由 AI 处理并自动点击选项。
    * **【✍️ 2. 精答主观题】**：自动筛选简答、填空等无选项题目，逐题请求 AI 并自动填入文本框。
4. 观察面板底部的**黑色实时日志窗**，等待提示“任务全部结束”后，人工核对一遍即可交卷。

## ⚙️ 常见问题 (FAQ)

**Q: 第一次运行提示“网络请求失败”或被拦截怎么办？**
**A:** 这是因为脚本需要跨域请求 AI 接口。在首次点击按钮时，油猴会在页面顶部弹出跨域警告（XHR Security）。请务必点击 **“总是允许 (Always Allow)”**。脚本已在 `@connect` 中配置了域名白名单，授权一次后即可永久放行。

**Q: 为什么点击客观题时提示“JSON被截断”？**
**A:** 当单页客观题数量极大（例如一次性提交上百道题），超出了大模型单次输出的最大 Token 限制时会触发此警告。脚本已内置**抢救机制**，会自动为您填写已经生成的答案，剩余未做完的题目请手动完成或分批处理。
